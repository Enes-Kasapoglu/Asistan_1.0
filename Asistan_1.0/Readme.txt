İlk önce gerekli modülleri indiriyoruz bunlar için ise terminal ekranına:

    $ pip install gtts
    $ pip install playsound
    $ pip install sqlite3
    $ pip install feedparser

Daha sonra dosyayı çalıştırabiliriz.

1. Adım

Çalıştırdıktan sonra ilk önce kendi bilgilerinizi tanıtmanız gerekir bunun için ise:

    # 'bilgilerimi kaydet' veya 'Bilgilerimi kaydet'

Yazıp, bilgilerinizi girerek kaydedebilirsiniz.


2.Adım

Daha sonra ise asistan'a vermek istediğiniz özellikleri yazabilirsiniz. Bunun için ise:

    # 'asistan bilgilerini kaydet' veya 'Asistan bilgilerini kaydet'

Yazabilirsiniz.


3. Adım Son

Komut listesi belgesini açarsanız kullanabileceğiniz komutlar orada belirtilmiştir. Bu komutlar ile asistanı kullanabilirsiniz.
